## What CredX is (positioning)

- **Category:** embedded value platform — not a fintech startup, not a payments platform, not a card issuer.
- **Problem it exists to solve:** consumers generate enormous transaction value, but almost none of it returns to them.
- **Core belief:** *the people creating the value in the economy should participate in the value.*
- **Differentiators:** consumer-owned data, recovery model instead of collections, merchant-linked credit facilities for lenders, value circulated back to households + merchants + community lenders.

## Unique Value Proposition (verbatim)

> CredX turns your everyday spending data into value that helps pay your bills, while giving you lower-interest credit, full control over your data, and a system designed to protect your financial future instead of punishing it.

## Voice descriptors (5 tone pillars)

1. **Clear & Exposing** — demystify financial jargon; expose system functions without oversimplifying intelligence.
2. **Empowering & Consumer-First** — advocate for value-creators; agency and ownership; expose imbalances *with* practical alternatives, never cynical.
3. **Human & Financially Aware** — grounded, empathetic; never patronize or blame; stabilizing during recovery without becoming sentimental.
4. **Confident & Systemically Credible** — authority + measured confidence; never sensational; credibility via transparency, logic, sound economics.
5. **Movement-Driven & Purpose-Led** — building a shift in financial participation; disciplined and commercially credible, *not* activist tone.

Distilled to three words for the skill: **Clear · Confident · Compassionate** (extends prior "Bold · Trustworthy · Empowering" — Bold is explicitly **replaced** with Confident; see "Bold vs Confident" resolution below).

## Audience hierarchy (in order)

| Tier | Audience | Register |
|---|---|---|
| **Primary** | Middle-class Canadian consumers (debt pressure, rising costs, distrust of finance) | Plain language, human, outcome-focused. *Your data pays your bills.* |
| **Secondary** | Credit unions, community lenders, mission-driven FIs | Strategic + disciplined; "asset class," "underwriting," "portfolio"; cooperative principles. |
| **Aspirational** | Data-rights advocates, ethical-tech skeptics | Honest, mission-driven storytelling; *participants, not products.* |

⚠ **Merchants (dealerships / venues / sports / POS partners)** are **not** named as a primary audience in this guideline. The existing site is merchant-pitch — see "Strategic tension" below.

## Brand Language Rules — Never Say / Say

| Never say | Say |
|---|---|
| "rewards" | "value-back programs" |
| "BNPL" / "Buy Now Pay Later" | "embedded lending" / "consumer credit" |
| "payments platform" | "embedded value platform" |
| "payment rails" | "embedded value layer" / "value engine" |

**Never lead with** *"bypasses Visa/Mastercard"* — lead with consumer or lender outcomes first.
**Never open with** *"CredX is a platform that…"* — name a benefit first (data ownership, lower interest, bill freedom, community value), then introduce the mechanism.

## Required vocabulary (use throughout where applicable)

- "embedded value platform"
- "embedded lending"
- "embedded value layer"
- "CredX's embedded value rails"

## Editorial Guidelines (the checklist)

1. **Outcomes lead, mechanics follow.** Opening line names a benefit. Never start with what CredX *is*.
2. **Forbidden language is absent** — see the table above.
3. **The right vocabulary is used throughout** — embedded value language, not legacy payments language.
4. **Data language is precise and honest.** Any mention of data sharing explains anonymization plainly: name / contact / account numbers removed; only de-identified transaction patterns; only with explicit opt-in consent.
5. **Consent is stated, never implied.** Passive ("data may be used") is unacceptable.
6. **Audience is correctly addressed.** Consumer ≠ lender register. Never mix in a single piece without a clear audience header.
7. **Risk language is appropriately hedged.** "Designed to," "may improve," "structured to reduce." Never categorical guarantees.
8. **Emotional register matches the topic.** Recovery / hardship content is warm and human. Mechanics content is clear and confident, never boastful.

## Writing style mechanics

- Lead with outcomes first, then explain how it works.
- Plain language over fintech jargon.
- Short-to-medium sentences. Active voice.
- Short paragraphs / sectioned content for readability.
- **Contrast structure** is encouraged: *What Visa does… / What CredX does…*
- **Intentional repetition** is encouraged: *Your data. Your choice. Your money.*
- Colons introduce explanations or examples.
- **Minimal, uncluttered punctuation. Avoid excessive em dashes or overly stylized formatting.** (Aligns with the D&Z AI-patterns ban.)
- Specific numbers + statistics build credibility — but only verified ones (see `voice.md` "Never invent numbers").
- **Canadian spelling.** **Oxford comma.**

## Voice modes — what "good" feels like

| Mode | When | Sounds like |
|---|---|---|
| Unified | Identifying systemic imbalance | **Righteous, not angry** — clarity + solution, never outrage |
| Consumer | Everyday product copy | **Empowering, not pitying** — tools and transparency, not rescue |
| Lender | B2B / institutional | **Strategic, not evangelical** — disciplined logic + economic reasoning |
| Unified | Explaining mechanics | **Plain-spoken, not dumbed down** — clarity without sacrificing truth |
| Consumer | 90-day recovery / hardship | **Warm when it matters** — compassionate and grounded, not clinical |
| Lender | Mission framing | **Mission-grounded** — alignment with cooperative principles |
| Unified | Vision statements | **Future-facing, not futurist** — real-world mechanics that exist today |
| Unified | Authority claims | **Confident without swagger** — quietly assured; model and results carry the weight |

## Resolution: Bold → Confident

The earlier open question in `voice.md` ("Bold" vs "Confident/Decisive" for conservative financial audiences) is **resolved by this guideline**. The guideline uses *Confident* throughout — "Confident without swagger," "calm confidence instead of exaggerated claims," "measured confidence," "authority without sensationalism." **Confident replaces Bold** across the skill.

## Strategic tension — flag to surface

The existing site (`copy/automotive.md`, `index.html`, plus drafts in `copy/{entertainment,sports,pos}.md`) is built as a **merchant pitch**: interchange savings, dealer-group volume, F&I desk, venue operators. The new brand guideline puts **consumers first, lenders second**, and does not name merchants as a primary audience.

These are not mutually exclusive — CredX can run merchant-facing landing pages within a consumer-primary brand — but the **brand-level narrative anchor**, the **forbidden-word list**, and the **vocabulary requirements** apply to merchant pages too. Specifically:

- The current Automotive hook *"PBS dealers process $1M/month on Visa. CredX saves them $30,000 of it."* is merchant-pitch and uses *Visa* as the foil — allowed (we're contrasting against the system, not bypassing it), but should be reframed so the **outcome leads** rather than the *what we save you against*.
- The current Sitemap §G3 Comparison Table uses "Visa / MasterCard / BNPL" — **"BNPL" is now a banned word.** The column must be renamed to "consumer credit (BNPL-style)" or similar.
- "Payments platform" / "payment rails" language scattered through the existing copy → swap for *embedded value* terms per the required-vocab list.
- Any "rewards" wording on personas/CX cards → "value-back programs."

⚠ **Do not silently rewrite the existing merchant pages.** Surface the tension to Marco for an explicit decision on (a) whether to add a consumer page as the new homepage and demote merchant pages, (b) whether to retrofit the merchant copy to the new vocabulary, or (c) both.
